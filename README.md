AET_HARMONY — Libro Sagrado

Creado por cb4s_ | LQISTUDIO

AET_HARMONY es un idioma creado como sistema de cifrado para transformar textos, números, notas musicales y símbolos en una estructura lógica, coherente y organizada.
Este repositorio contiene la documentación oficial del idioma, incluyendo su códice, normas de escritura y metodología.

Apéndice con símbolos clave

Toda la estructura presentada en el Libro Sagrado está construida para mantener coherencia gramatical, simbólica y funcional al momento de cifrar o descifrar.

Firma del creador

La firma oficial del creador corresponde a una abreviatura cifrada de LQI dentro del sistema AET_HARMONY.
Cada letra se basa en el códice:

L → λ-ξ
Q → θ-ψ
I → ι-κ

La firma puede representarse como un glifo único derivado de estos símbolos, aplicando las reglas de abreviación del idioma.
